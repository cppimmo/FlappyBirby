# Flappy Birby (C SDL2 Flappy Bird)
Made by cppimmo 8/1/20 in **C99** with **SDL2**
# Dependencies
    SDL2
    SDL2_image
    SDL2_mixer
    SDL2_ttf
# Building
The CMakeLists.txt hasn't been completed instead use make:
```
$ make
```
This will place a debug and release build in the build/ directory.
